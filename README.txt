# Fontionnement
Lancer le programme identification.py qui permettre de reconnaître les différentes personnes présente sur une vidéo.
Les noms des différentes personnes s'afficheront juste à côté du visage.

# Trainner
Le dossier trainner comporte les différentes models d'entraînement obtenue après execution du fichier apprentissages.py

05 modèls sont présent, suite à la classification des images manuellement obtenue après capture sur cette video : 
https://www.youtube.com/watch?v=qP6g6UZjOVY&t=1938s

# Identification

Pour lancer le programme il suffit de remplacer le chemin actuel cap=cv2.VideoCapture("video/video1.mp4"), par 
celui la video presente sur votre machine et se rassurer que le dossier trainner se trouve dans le même repertoire 
que le programme identification
